# sistema-monitoreo-temperatura
crear solución de temperatura en escuelas para monitorear la temperatura de las auas en las que se encuentrn los estudiantes
y de esta manera dar amntenimiento a los aires acondicionados y evitar que se dañen por falta de mantenimiento, además de garantizar un ambiente adecuado para los estudiantes. El sistema incluirá sensores de temperatura, una plataforma de monitoreo en tiempo real y alertas para el personal de mantenimiento cuando se detecten temperaturas fuera del rango establecido.
